# wrap-near-contract-methods-const
const's for wrap.near smart contract methods


ℹ️ can be used with any near api js option.


---


### DEV
to devlop and publish this package
```bash
bun install
bun run index.ts
bun build index.ts
bun publish --dry-run
bunx npm login
bun publish --access public
```

---

### HOW TO USE

add
```sh
npm i @sleet-js/wrap-near-contract-methods-const
bun add @sleet-js/wrap-near-contract-methods-const
```

import
```js
import { wrap_near_methods_const } from '@sleet-js/wrap-near-contract-methods-const';
```
